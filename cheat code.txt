git init -> setup git repository
git config --global user.name "nama anda" -> konfigur nama
git config --global user.email "email anda" ->konfigur email
git status -> periksa status fail dalam folder repository/projek
git add "nama fail" -> masukkan fail dalam stage
git commit -m "masukkan mesej" ->bagi tau apa perubahan yang dibuat.
git branch "nama branch" -> buat cawangan/branch baru
git checkout "nama branch" -> pergi ke direktori branch
git rm --cache"nama fail" -> untuk keluar dari stage
git add -all/-A -> add nama fail
